## US-001: L1 轻度预警（单指标减仓）

**As a** 情绪周期交易者  
**I want** 当单个风险指标触发时收到减仓预警  
**So that** 不再因AND逻辑半触发而陷入灰色地带无信号可依

### 验收标准

- [ ] AC-1.1: 涨跌比 < 50%（即使龙头未断板）时，系统输出L1减仓信号
- [ ] AC-1.2: 龙头质量降至C级（炸板6-10次）时，系统输出L1减仓信号
- [ ] AC-1.3: 涨跌比日内跌幅 > 30pp时，系统输出L1减仓信号
- [ ] AC-1.4: L1信号包含：⚠️标识 + 触发原因 + 具体数值 + "减仓至50%"动作指令
- [ ] AC-1.5: 用04-17 10:00数据验证：涨跌比26.5%<50% → L1触发 ✅（旧逻辑沉默 ✗）

### 故事点: 5  
### 优先级: P0

## US-002: 龙头封板质量评分

**As a** 情绪周期交易者  
**I want** 系统自动评估龙头封板质量（A/B/C/D四级）  
**So that** 能区分"强势封板的7板"和"14次炸板苟活的7板"

### 验收标准

- [ ] AC-2.1: 每个采集时点自动计算龙头质量等级（A/B/C/D）
- [ ] AC-2.2: 炸板>10次的龙头被评为D级
- [ ] AC-2.3: D级龙头在止损判定中视同断板处理
- [ ] AC-2.4: 用04-17圣阳14次炸板验证：质量=D → 视同断板 → L2可触发
- [ ] AC-2.5: MVP阶段仅用炸板次数评分，封单金额作为P1迭代项

### 故事点: 5  
### 优先级: P0

## US-003: L2 中度警报（多指标清仓）

**As a** 情绪周期交易者  
**I want** 当多个风险指标共振或趋势确认恶化时收到清仓信号  
**So that** 能在趋势明确转坏时果断出清，不被龙头"名义未断板"迷惑

### 验收标准

- [ ] AC-3.1: 组合A（涨跌比<50% + 龙头断板/质量D）触发清仓
- [ ] AC-3.2: 组合B（涨跌比<35% + 龙头质量≤C）触发清仓
- [ ] AC-3.3: 组合C（连续2时点涨跌比<40%）触发清仓
- [ ] AC-3.4: 组合D（3板以上连板股<3只）触发清仓
- [ ] AC-3.5: L2触发后进入冷却期（≥1个交易日）
- [ ] AC-3.6: 用04-17数据回测：组合A/B均在10:00触发（旧AND全天未触发）

### 故事点: 5  
### 优先级: P0

## US-004: L3 极端警报（紧急清仓）

**As a** 情绪周期交易者  
**I want** 在市场崩盘级极端行情下获得即时清仓指令  
**So that** 不需要等待多指标确认就能第一时间保护本金

### 验收标准

- [ ] AC-4.1: 涨跌比<15%时单指标即触发L3
- [ ] AC-4.2: 跌停家数≥20时单指标即触发L3
- [ ] AC-4.3: 总龙头跌停时单指标即触发L3
- [ ] AC-4.4: L3触发后进入强制冷却期（≥2个交易日）
- [ ] AC-4.5: L3触发后自动进入v1.1冰点观察模式

### 故事点: 3  
### 优先级: P0

## US-005: 灰色地带消除

**As a** 情绪周期交易者  
**I want** 在任何市场状态下都能收到明确的操作指令（而非沉默）  
**So that** 不再凭感觉在半触发时做决策

### 验收标准

- [ ] AC-5.1: 涨跌比<50%但龙头未断板 → L1减仓（不再沉默）
- [ ] AC-5.2: 涨跌比<50%但龙头质量D → L2清仓（不再视为未断板）
- [ ] AC-5.3: 龙头断板但涨跌比>50% → L1减仓
- [ ] AC-5.4: 用04-15/16/17三日数据覆盖测试，决策流程图无死角
- [ ] AC-5.5: 盘中止损级别只升不降，降级需隔日确认

### 故事点: 3  
### 优先级: P0

## US-006: 恢复条件与冷却期

**As a** 情绪周期交易者  
**I want** 止损触发后有明确的恢复入场条件和冷却期  
**So that** 不会在V型反转时过早复位，也不会因冷却期过长错过真正的反转

### 验收标准

- [ ] AC-6.1: L1恢复需：涨跌比>55%持续1h + 龙头质量≥B
- [ ] AC-6.2: L2恢复需：冷却1日 + 涨跌比>60%（2时点确认）+ 龙头≥B + 非尾盘拉升
- [ ] AC-6.3: L3恢复需：冷却2日 + 涨跌比>65%全天 + 涨停>50 + 新龙头A级
- [ ] AC-6.4: 用04-16 V型反转数据验证：恢复信号不早于12:00
- [ ] AC-6.5: 冷却期内即使恢复条件满足也不提前恢复

### 故事点: 3  
### 优先级: P1
