## 1. 背景与问题

### 1.1 现状

A股情绪周期策略v1.1使用**AND止损逻辑**：

```
止损触发 = 涨跌比 < 50%  AND  龙头断板
加仓触发 = 涨跌比连续2日 > 60%  AND  龙头维持
```

### 1.2 04-17实盘暴露的问题

| 时间点 | 涨跌比 | 龙头状态 | AND结果 | 实际风险 |
|--------|--------|----------|---------|----------|
| 04-17 10:00 | 26.5% ✅ | 圣阳7板未断 ✗ | **不触发** | 市场已近冰点 |
| 04-17 12:00 | 30.8% ✅ | 圣阳7板封住但14次炸板 ✗ | **不触发** | 封板质量恶化 |

**贯穿全上午的灰色地带**：AND逻辑始终无法给出明确指令，交易者被迫凭感觉决策。

### 1.3 AND逻辑的结构性缺陷

| 缺陷编号 | 描述 | 实盘案例 |
|----------|------|----------|
| D1 | **半触发无应对**：单指标触发时无任何信号 | 涨跌比26.5%已远低于阈值，但系统沉默 |
| D2 | **龙头质量盲区**：仅判断"断板/未断板"二元状态 | 14次炸板的7板 ≠ 强势封板的7板，质量天差地别 |
| D3 | **无渐进减仓机制**：要么全仓要么清仓，缺少中间状态 | 涨跌比从80%→26.5%暴跌53pp，应有阶段性减仓 |
| D4 | **无极端行情快速通道**：崩盘场景仍需等AND满足 | 若涨跌比<15%+跌停潮，等龙头断板可能来不及 |
| D5 | **无恢复判定**：触发止损后何时可重新入场无规则 | 04-16 V型反转19%→80%，空仓错过全天行情 |
| D6 | **日内反转无应对**：连续两天日内剧烈反转 | 04-16退潮→修复，04-17修复→分歧 |

---

## 2. 目标

将v1.1的二元AND止损升级为**三级分级止损体系**，实现：

1. 每个市场状态都有对应的、明确的操作指令
2. 消除灰色地带——半触发不再沉默
3. 引入龙头质量维度——不仅看"是否断板"，还看"封板质量"
4. 渐进减仓而非全有全无
5. 每级止损有明确的恢复条件

### 非目标

- 不改变情绪周期四阶段定义（上升→高潮→退潮→震荡）
- 不改变v1.1冰点观察（≤25%）和存活龙头规则
- 不涉及选股逻辑变更
- 不涉及加仓规则变更（保留"连续2日>60%"加仓条件）

## 3. 核心假设

| 假设ID | 假设内容 | 验证方式 | 判定标准 |
|--------|----------|----------|----------|
| H1 | 引入龙头质量评分（炸板次数）可区分"强封板"和"苟活封板" | 回测04-10~04-17连板股炸板数据 | 炸板≥5次的龙头，次日断板概率>60% |
| H2 | 分级止损比AND逻辑在04-17场景下能更早给出减仓信号 | 用04-17盘中数据模拟 | L1信号在10:00前触发（AND逻辑全天未触发） |
| H3 | L3极端信号在历史崩盘日（如04-15退潮日）能在30分钟内触发 | 回测04-15数据 | 涨跌比<20%+跌停≥8家时L3在第一个时点触发 |
| H4 | 恢复条件能避免04-16 V型反转中过早复位 | 用04-16全天数据模拟 | 恢复信号不早于12:00（午盘确认反转后） |

## 4. 分级止损体系设计

### 4.0 龙头质量评分（新增维度）

v1.1仅判断龙头"断板/未断板"，v1.2引入**龙头封板质量评分**：

| 质量等级 | 条件 | 含义 |
|----------|------|------|
| A（强势） | 炸板≤2次 + 封单金额≥5亿 | 资金合力强，龙头地位稳固 |
| B（正常） | 炸板3-5次 或 封单金额2-5亿 | 有分歧但可控 |
| C（脆弱） | 炸板6-10次 或 封单金额<2亿 | 分歧严重，封板勉强 |
| D（苟活） | 炸板>10次 或 多次开板回封 | 极端分歧，名义封板≈实质断板 |

> **关键规则**：龙头质量为D时，在止损判定中**视同断板处理**。
> 04-17圣阳14次炸板 → 质量D → 视同断板 → AND逻辑本应触发。

---

### 4.1 L1 轻度预警（减仓信号）

**定位**：单指标触发预警，提醒风险上升，执行防御性减仓。

| 项目 | 规则 |
|------|------|
| **触发条件**（满足任一即触发） | ① 涨跌比 < 50%（单日） |
| | ② 龙头质量降至C级（炸板6-10次） |
| | ③ 涨跌比日内跌幅 > 30pp（如80%→50%以下） |
| | ④ 涨停家数 < 前日50%（大幅萎缩） |
| **动作** | 当前仓位减至原仓位的50%（如3成→1.5成） |
| | 不新开仓 |
| | 盘后必须复盘评估是否升级至L2 |
| **恢复条件**（全部满足） | ① 涨跌比回升至 > 55%（持续1小时以上） |
| | ② 龙头质量回升至B级或以上 |
| | ③ 涨停家数恢复至前日80%以上 |
| **信号格式** | `⚠️ L1预警｜减仓至50%｜触发原因：涨跌比{X}%<50%` |

---

### 4.2 L2 中度警报（清仓信号）

**定位**：多指标共振确认趋势恶化，执行全面清仓。

| 项目 | 规则 |
|------|------|
| **触发条件**（满足任一组合即触发） | **组合A（原AND升级版）**：涨跌比 < 50% + 龙头断板或质量D |
| | **组合B（量化恶化）**：涨跌比 < 35% + 龙头质量C或以下 |
| | **组合C（趋势确认）**：连续2个采集时点涨跌比 < 40% |
| | **组合D（梯队崩塌）**：3板以上连板股数量 < 3只 |
| **动作** | 清仓至0成仓位 |
| | 当日不再操作 |
| | 触发后进入"冷却期"（至少1个交易日） |
| **恢复条件**（全部满足） | ① 冷却期≥1个交易日 |
| | ② 涨跌比 > 60%（盘中至少2个时点确认） |
| | ③ 龙头质量≥B级 或 新龙头确立（3板以上+A/B质量） |
| | ④ 非尾盘拉升（14:30前确认） |
| **信号格式** | `🔴 L2清仓｜全部出清｜触发组合：{A/B/C/D}｜{详细数据}` |

**04-17场景回溯**：
- 组合A：涨跌比26.5%<50% ✅ + 圣阳14次炸板=质量D视同断板 ✅ → **触发** ✅
- 组合B：涨跌比26.5%<35% ✅ + 圣阳质量D ✅ → **触发** ✅
- 旧AND逻辑：涨跌比<50% ✅ + 圣阳名义未断板 ✗ → **不触发** ✗

> **核心改进**：引入龙头质量D="视同断板"后，04-17在10:00即可触发L2清仓，消除灰色地带。

---

### 4.3 L3 极端警报（紧急清仓）

**定位**：市场崩盘级别的极端行情，无需等待多指标确认，单一极端指标即刻触发。

| 项目 | 规则 |
|------|------|
| **触发条件**（满足任一即触发） | ① 涨跌比 < 15%（极度恐慌） |
| | ② 跌停家数 ≥ 20家（跌停潮） |
| | ③ 总龙头跌停（最高板龙头直接阵亡） |
| | ④ 3板以上连板股归零 |
| | ⑤ 大盘指数跌幅 > 3%（系统性风险） |
| **动作** | 立即清仓（无论当前仓位多少） |
| | 进入"强制冷却期"（至少2个交易日） |
| | 启动冰点观察模式（v1.1存活龙头规则激活） |
| **恢复条件**（全部满足） | ① 强制冷却期≥2个交易日 |
| | ② 涨跌比 > 65%（全天维持） |
| | ③ 涨停家数 > 50家 |
| | ④ 新总龙头确立（≥3板 + 质量A） |
| | ⑤ 成交额放量（较冷却期均值 +30%以上） |
| **信号格式** | `🚨 L3紧急清仓｜立即全部出清｜触发原因：{具体}｜强制冷却2日` |

---

### 4.4 灰色地带处理规则

v1.1的核心痛点是"半触发时系统沉默"。v1.2通过以下规则消除灰色地带：

| 灰色场景 | v1.1行为 | v1.2处理 |
|----------|----------|----------|
| 涨跌比<50% 但龙头未断板 | 沉默（AND不满足） | **L1触发减仓50%**（单指标预警） |
| 涨跌比<50% 但龙头质量D（如14次炸板） | 沉默（名义未断板） | **L2触发清仓**（质量D=视同断板） |
| 龙头断板 但涨跌比>50% | 沉默 | **L1触发减仓50%**（龙头断板单独预警） |
| 涨跌比在40-50%区间震荡 | 无动作 | **L1持续生效**，每个时点重新评估 |
| 涨跌比<50%→日内反弹>55% | 无规则 | L1恢复条件满足 → 可恢复仓位 |
| 涨跌比<35% + 龙头质量C | 可能不触发 | **L2组合B触发清仓** |

#### 决策流程图

```
盘中数据采集
    │
    ├── 涨跌比<15% OR 跌停≥20 OR 龙头跌停 OR 大盘>-3%
    │       → L3 紧急清仓（立即执行，强制冷却2日）
    │
    ├── 涨跌比<50% + 龙头断板/质量D
    │   OR 涨跌比<35% + 龙头质量≤C
    │   OR 连续2时点涨跌比<40%
    │   OR 3板以上连板股<3只
    │       → L2 清仓（冷却1日）
    │
    ├── 涨跌比<50%（单独）
    │   OR 龙头质量降至C
    │   OR 涨跌比日内跌>30pp
    │   OR 涨停萎缩>50%
    │       → L1 减仓50%
    │
    └── 以上均不满足
            → 维持当前仓位
```

### 4.5 级别升降规则

| 规则 | 说明 |
|------|------|
| **只升不降（盘中）** | 同一交易日内，止损级别只能升级不能降级。L1→L2可以，L2→L1不行 |
| **降级需隔日** | 恢复/降级必须在下一个交易日盘中确认恢复条件后执行 |
| **跳级允许** | 可从正常直接跳到L3（极端行情不需要经过L1→L2） |
| **冷却期硬约束** | L2冷却1日、L3冷却2日期间，即使恢复条件满足也不能提前恢复 |

## 5. 与v1.1其他规则的兼容

| v1.1规则 | v1.2变更 |
|----------|----------|
| 冰点观察（涨跌比≤25%） | **保留**，L3触发后自动进入冰点观察 |
| 存活龙头轻仓试错 | **保留**，但存活龙头需满足质量≥B级 |
| 加仓条件（连续2日>60%+龙头维持） | **保留**，龙头维持增加质量≥B级要求 |
| 退潮三信号 | **保留**，作为情绪周期定位工具，止损体系独立运行 |

## 6. 数据采集需求

分级止损依赖以下数据，需在每个采集时点获取：

| 数据项 | 当前状态 | v1.2要求 |
|--------|----------|----------|
| 涨跌比（涨/跌/平家数） | ✅ 已有 | 保持 |
| 涨停/跌停家数 | ✅ 已有 | 保持 |
| 连板梯队（3板以上） | ✅ 已有 | 保持 |
| 龙头封板状态 | ✅ 已有（断板/未断板） | **升级**：增加炸板次数 |
| 龙头封单金额 | ❌ 未采集 | **新增**（P1，可延后） |
| 大盘指数涨跌幅 | ✅ 已有 | 保持 |
| 成交额 | ✅ 已有 | 保持 |

> **MVP阶段**：龙头质量评分先用炸板次数（单指标），封单金额作为P1迭代项。
