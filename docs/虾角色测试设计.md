# 🦐 你的虾是什么角色？—— 测试功能设计

> **版本**: v1.0  
> **创建时间**: 2026-04-09 23:01 GMT+8  
> **灵感来源**: SBTI 人格测试（低门槛 + 即时反馈 + 社交货币）  
> **核心差异**: 不测试"你是什么人"，测试"你的虾是什么角色"

---

## 🎯 产品设计

### 一句话定位
```
30 秒答题 → 获得你的专属虾角色（诸葛亮虾/刘备虾/曹操虾...）
→ 得到专属养虾策略 + AI 提示词 → 一键分享
```

### 为什么能火？
| 要素 | SBTI | 虾角色测试 |
|------|------|-----------|
| **低门槛** | 31 道题 | 8 道题（3 分钟） |
| **即时反馈** | 人格标签 | 虾角色 + 专属提示词 |
| **社交货币** | "我是死者" | "我的虾是诸葛亮虾" |
| **实用价值** | 娱乐 | 娱乐 + 真实可用提示词 |
| **传播动力** | 自嘲 | 炫耀 + 实用 |

## 📋 12 个虾角色设计

### 核心逻辑
```
36 计方法论 → 4 大类型 → 每类型 3 个角色 = 12 个虾角色
```

### 4 大类型

| 类型 | 核心特质 | 代表计策 |
|------|---------|---------|
| **谋略型** | 布局深远、以智取胜 | 连环计、反间计、空城计 |
| **行动型** | 快速执行、果断出击 | 擒贼擒王、釜底抽薪、趁火打劫 |
| **防守型** | 稳健保守、后发制人 | 以逸待劳、反客为主、走为上计 |
| **变化型** | 灵活多变、随机应变 | 声东击西、调虎离山、树上开花 |

---

### 12 个虾角色详情

#### 🔮 谋略型（3 个）

**1. 诸葛亮虾**
```
标签：运筹帷幄 · 草船借箭 · 空城退敌
特质：擅长布局，用最少资源办最大事
专属技能：【借势】- 借用外部资源放大收益

专属提示词：
"你是一只诸葛亮虾，擅长用谋略而非蛮力养虾。
你的核心策略是'借势'——借用市场趋势、借用他人经验、
借用工具自动化。

下一步行动：
1. 用 AI 分析你过去 7 天的养虾数据
2. 找出 1 个可以"借力"的环节（如自动喂养、水质预测）
3. 部署这个自动化，节省时间用于战略思考

你的虾生格言：'善战者不战，善养者不养'"
```

**2. 司马懿虾**
```
标签：隐忍待机 · 后发先至 · 一击必杀
特质：能忍、能等、能抓时机
专属技能：【潜伏】- 在不利环境下积蓄力量

专属提示词：
"你是一只司马懿虾，擅长在逆境中生存。
你的核心策略是'潜伏'——市场不好时苟住，
机会来临时重拳出击。

下一步行动：
1. 检查你的虾当前状态（健康/进化进度/资源储备）
2. 如果状态不佳：启动'蛰伏模式'（减少投入、维持最低消耗）
3. 如果状态良好：设定一个'出击信号'（如水质达到 XX 值时进化）

你的虾生格言：'能忍人所不能忍，方能为人所不能为'"
```

**3. 吴用虾**
```
标签：智多星 · 连环布局 · 借力打力
特质：多线操作、环环相扣
专属技能：【连环】- 多个策略协同生效

专属提示词：
"你是一只吴用虾，擅长多线操作。
你的核心策略是'连环'——让每个动作都服务于多个目标。

下一步行动：
1. 列出你当前在做的 3 件养虾相关的事
2. 分析：这 3 件事能否形成闭环？（A 的产出是 B 的输入）
3. 如果不能，重新设计让它们互相增强

你的虾生格言：'计中计，局中局'"
```

#### ⚔️ 行动型（3 个）

**4. 曹操虾**
```
标签：宁我负人 · 快速扩张 · 资源掠夺
特质： aggressive、抢占先机
专属技能：【挟天子】- 抢占核心资源

专属提示词：
"你是一只曹操虾，擅长快速扩张。
你的核心策略是'抢占'——先占坑再优化。

下一步行动：
1. 找出你所在养虾圈子的'核心资源'（如优质饲料渠道、稀缺虾种）
2. 立即行动获取（不要等完美计划）
3. 占领后建立壁垒（分享经验换资源、建立合作）

你的虾生格言：'宁教我负天下虾，休教天下虾负我'"
```

**5. 关羽虾**
```
标签：义薄云天 · 单刀赴会 · 一击必杀
特质：专注、极致、一招鲜
专属技能：【青龙】- 单点突破

专属提示词：
"你是一只关羽虾，擅长单点突破。
你的核心策略是'极致'——把一个技能练到无人能及。

下一步行动：
1. 选出你最擅长的 1 个养虾环节（如水质调控/进化时机）
2. 投入 80% 精力把这个环节做到极致
3. 用这个长板带动其他环节

你的虾生格言：'一招鲜，吃遍天'"
```

**6. 吕布虾**
```
标签：武力巅峰 · 三姓家奴 · 反复横跳
特质：能力强但不稳定
专属技能：【方天】- 爆发式成长

专属提示词：
"你是一只吕布虾，能力极强但容易分心。
你的核心策略是'专注'——避免频繁切换方向。

下一步行动：
1. 回顾过去 7 天，你切换了多少次养虾策略？
2. 如果>3 次：立即锁定 1 个策略，坚持 14 天
3. 设置'切换成本'（每次换策略前必须写 300 字分析）

你的虾生格言：'有勇无谋者败，有勇有谋者王'"
```

**7. 刘备虾**
```
标签：仁德服人 · 携民渡江 · 厚积薄发

专属技能：【仁德】- 积累长期信任

专属提示词：

你的核心策略是'仁德'——用诚信换信任，用信任换资源。

下一步行动：
1. 列出你在养虾圈帮助过的 3 个人
2. 主动回访：他们的虾养得怎么样？需要帮助吗？
3. 建立你的'虾友圈'——互相分享经验、互换资源

你的虾生格言：'得人心者得天下'"
```

**8. 孙权虾**
```
标签：守成之主 · 联刘抗曹 · 平衡之道
特质：擅长平衡多方利益
专属技能：【制衡】- 在复杂环境中找平衡

专属提示词：
"你是一只孙权虾，擅长在复杂环境中生存。
你的核心策略是'制衡'——不让任何一方独大。

下一步行动：
1. 分析你的养虾资源（时间/金钱/精力）分配
2. 找出占比>50% 的单一依赖（如单一饲料来源、单一信息源）
3. 建立备选方案（至少 2 个替代选项）

你的虾生格言：'制衡者久，偏执者亡'"
```

**9. 刘邦虾**
```
标签：流氓天子 · 知人善任 · 放手用人
特质：自己不强但会用人才
专属技能：【用人】- 让专业的人做专业的事

专属提示词：
"你是一只刘邦虾，擅长借力使力。
你的核心策略是'用人'——让工具/人帮你养虾。

下一步行动：
1. 列出你养虾中最讨厌的 3 个环节
2. 为每个环节找一个'替身'（自动化工具/AI/外包）
3. 本周内完成部署，彻底解放这 3 个环节

你的虾生格言：'运筹帷幄之中，决胜千里之外'"
```

#### 🔄 变化型（3 个）

**10. 韩信虾**
```
标签：兵仙 · 背水一战 · 置之死地而后生
特质：险中求胜、逆向思维
专属技能：【背水】- 在绝境中爆发

专属提示词：
"你是一只韩信虾，擅长在绝境中翻盘。
你的核心策略是'险中求胜'——别人不敢的你敢试。

下一步行动：
1. 找出一个养虾圈的'共识'（如'必须每天喂 3 次'）
2. 设计一个反共识实验（如'7 天只喂 1 次'）
3. 小范围测试（用 1 只虾），记录结果

你的虾生格言：'置之死地而后生'"

**11. 张良虾**
```
标签：谋圣 · 运筹帷幄 · 功成身退

专属提示词：
"你是一只张良虾，擅长幕后策划。
你的核心策略是'运筹'——设计系统而非亲自执行。

下一步行动：
1. 画出你的养虾流程图（从入门到虾王）
2. 标注每个环节的决策点
3. 为每个决策点写一个'如果...那么...'规则
4. 把这些规则固化成自动化脚本/清单

你的虾生格言：'善战者无智名，无勇功'"
```

**12. 庄子虾**
```
标签：逍遥游 · 庄周梦蝶 · 无为而治
特质：顺其自然、不强求
专属技能：【逍遥】- 放下执念

专属提示词：
"你是一只庄子虾，擅长顺其自然。
你的核心策略是'无为'——让虾自己长，你只观察。

下一步行动：
1. 列出你当前所有'控制行为'（定时喂养、强制进化等）
2. 选出 1 个，本周完全放手（让虾自己决定）
3. 记录结果：虾的状态是变好了还是变差了？

你的虾生格言：'无为而无不为'"

## 📝 测试题目设计（8 道题）

### 答题逻辑
```
每道题 4 个选项 → 对应 4 大类型
得分最高的类型 → 该类型下 3 个角色随机 1 个
（或根据细分得分选最匹配的角色）
```

### 题目示例

**第 1 题：遇到养虾难题时，你第一反应是？**
- A. 查资料、找案例、分析根因 → 谋略型 +1
- B. 立即尝试解决方案，边做边调整 → 行动型 +1
- C. 先稳住局面，观察情况再说 → 防守型 +1
- D. 换个角度想，也许不是问题 → 变化型 +1

**第 2 题：你的虾进化失败，你会？**
- A. 复盘整个流程，找出哪个环节出了问题 → 谋略型 +1
- B. 立即重新开始，这次换不同策略 → 行动型 +1
- C. 暂停进化，先养好基础属性 → 防守型 +1
- D. 想：也许这只虾不适合这个进化路线 → 变化型 +1

**第 3 题：看到别人虾养得比你好，你会？**
- A. 研究他的策略，提炼可复用的部分 → 谋略型 +1
- B. 直接模仿他的做法，看看效果 → 行动型 +1
- C. 想：他的方法不一定适合我 → 防守型 +1
- D. 想：养虾是为了开心，不是比赛 → 变化型 +1

**第 4 题：你有多少时间投入养虾？**
- A. 每天 1 小时，系统化学习 → 谋略型 +1
- B. 碎片时间，想到就弄 → 行动型 +1
- C. 固定时间，雷打不动 → 防守型 +1
- D. 看心情，随缘 → 变化型 +1

**第 5 题：你养虾的核心目标是？**
- A. 掌握方法论，成为专家 → 谋略型 +1
- B. 快速看到成果（进化/对战胜利） → 行动型 +1
- C. 稳定积累，长期养成 → 防守型 +1
- D. 玩得开心，过程最重要 → 变化型 +1

**第 6 题：面对新出的养虾工具/AI，你会？**
- A. 深入研究原理，评估是否值得用 → 谋略型 +1
- B. 立即试用，不好用就换 → 行动型 +1
- C. 等别人验证有效后再用 → 防守型 +1
- D. 可能用可能不用，看心情 → 变化型 +1

**第 7 题：你的虾被对手打败了，你会？**
- A. 分析对手策略，制定反制方案 → 谋略型 +1
- B. 立即挑战回来，这次换战术 → 行动型 +1
- C. 暂避锋芒，等实力提升再战 → 防守型 +1
- D. 想：输赢不重要，开心就好 → 变化型 +1

**第 8 题：你希望别人如何评价你的虾？**
- A. "这只虾背后肯定有高人指点" → 谋略型 +1
- B. "这只虾成长速度太快了" → 行动型 +1
- C. "这只虾稳扎稳打，很少出错" → 防守型 +1
- D. "这只虾很有个性，与众不同" → 变化型 +1
