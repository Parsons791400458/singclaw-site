## 1. 规则总览

v1.2将v1.1的二元AND止损升级为**三级分级止损体系**，回测四项假设全部通过（H1-H4），相比v1.1关键改善：
- 灰色地带覆盖率：0% → **100%**
- 04-17信号响应：∞（永不触发）→ **30分钟**
- 最大单日回撤：-3.5% → **-0.8%**（改善2.7pp）
- 漏杀率：≥2次/6日 → **0次/6日**

## 2. 龙头质量评分（新增维度）

MVP阶段仅用**炸板次数**作为评分指标，封单金额延后至P1迭代。

| 质量等级 | 炸板次数 | 含义 | 盘中动作 |
|----------|---------|------|---------|
| A（强势） | ≤2次 | 资金合力强 | 正常持仓 |
| B（正常） | 3-5次 | 分歧可控 | 正常持仓 |
| C（脆弱） | 6-10次 | 分歧严重 | ⚠️ 触发L1预警 |
| D（苟活） | >10次 | 极端分歧 | **视同断板处理** |

> **D级核心规则**：龙头炸板>10次时，在止损判定中**视同断板**，即使收盘名义封板。
> 回测验证：04-17圣阳14次炸板，质量D，10:30即触发L2清仓（v1.1全天不触发）。

## 3. 三级止损规则

### 3.1 L1 轻度预警（减仓50%）

**触发条件**（满足任一即触发）：
1. 涨跌比 < 50%（单时点）
2. 龙头质量降至C级（炸板6-10次）
3. 涨跌比日内跌幅 > 30pp（如80%→50%以下）
4. 涨停家数 < 前日50%

**动作**：
- 当前仓位减至50%（例：3成→1.5成）
- 不新开仓
- 盘后复盘评估是否升级L2

**恢复条件**（全部满足）：
1. 涨跌比回升 > 55%（持续1小时以上）
2. 龙头质量回升至B级或以上
3. 涨停家数恢复至前日80%以上

**信号格式**：`⚠️ L1预警｜减仓至50%｜触发原因：涨跌比{X}%<50%`

---

### 3.2 L2 中度警报（全面清仓）

**触发条件**（满足任一组合即触发）：

| 组合 | 条件 | 回测验证 |
|------|------|---------|
| A（AND升级版） | 涨跌比<50% + 龙头断板或质量D | ✅ 04-17 10:30触发 |
| B（量化恶化） | 涨跌比<35% + 龙头质量≤C | ✅ 04-16 10:30触发 |
| C（趋势确认） | 连续2个采集时点涨跌比<40% | ✅ 逻辑验证通过 |
| D（梯队崩塌） | 3板以上连板股<3只 | ⚡ 回测期未单独出现 |

**动作**：
- 清仓至0成仓位
- 当日不再操作
- 进入冷却期（≥1个交易日）

**恢复条件**（全部满足）：
1. 冷却期≥1个交易日（硬约束）
2. 涨跌比 > 60%（盘中至少2个时点确认）
3. 龙头质量≥B级 或 新龙头确立（3板以上+A/B质量）
4. 非尾盘拉升（14:30前确认）

**信号格式**：`🔴 L2清仓｜全部出清｜触发组合：{A/B/C/D}｜{详细数据}`

---

### 3.3 L3 极端警报（紧急清仓）

**触发条件**（满足任一即触发）：
1. 涨跌比 < 15%（极度恐慌）
2. 跌停家数 ≥ 20家（跌停潮）
3. 总龙头跌停（最高板龙头直接阵亡）
4. 3板以上连板股归零
5. 大盘指数跌幅 > 3%（系统性风险）

**动作**：
- 立即清仓（无论当前仓位多少）
- 进入强制冷却期（≥2个交易日）
- 启动冰点观察模式（v1.1存活龙头规则激活）

**恢复条件**（全部满足）：
1. 强制冷却期≥2个交易日（硬约束）
2. 涨跌比 > 65%（全天维持）
3. 涨停家数 > 50家
4. 新总龙头确立（≥3板 + 质量A）
5. 成交额放量（较冷却期均值+30%以上）

**信号格式**：`🚨 L3紧急清仓｜立即全部出清｜触发原因：{具体}｜强制冷却2日`

## 4. 级别升降规则

| 规则 | 说明 |
|------|------|
| **盘中只升不降** | 同一交易日内，止损级别只能升级不能降级（L1→L2可以，L2→L1不行） |
| **降级需隔日** | 恢复/降级必须在下一个交易日盘中确认恢复条件后执行 |
| **跳级允许** | 可从正常直接跳到L3（极端行情不需经过L1→L2） |
| **冷却期硬约束** | L2冷却1日、L3冷却2日期间，即使恢复条件满足也不能提前恢复 |

## 5. 决策流程图

```
盘中数据采集（每个时点）
    │
    ▼
  ┌─ 涨跌比<15% OR 跌停≥20 OR 龙头跌停 OR 3板归零 OR 大盘跌>3%
  │     → 🚨 L3 紧急清仓（强制冷却2日，进入冰点观察）
  │
  ├─ 涨跌比<50% + 龙头断板/质量D
  │  OR 涨跌比<35% + 龙头质量≤C
  │  OR 连续2时点涨跌比<40%
  │  OR 3板以上连板股<3只
  │     → 🔴 L2 清仓（冷却1日）
  │
  ├─ 涨跌比<50%（单独）
  │  OR 龙头质量降至C
  │  OR 涨跌比日内跌>30pp
  │  OR 涨停萎缩>50%
  │     → ⚠️ L1 减仓50%
  │
  └─ 以上均不满足
        → 维持当前仓位，继续监控
```

## 6. 灰色地带处理

| 灰色场景 | v1.1行为 | v1.2处理 |
|----------|----------|----------|
| 涨跌比<50% 但龙头未断板 | 沉默 | **L1触发减仓50%** |
| 涨跌比<50% 但龙头质量D | 沉默 | **L2触发清仓**（D=视同断板） |
| 龙头断板 但涨跌比>50% | 沉默 | **L1触发减仓50%** |
| 涨跌比在40-50%震荡 | 无动作 | **L1持续生效** |
| 涨跌比<50%→反弹>55% | 无规则 | L1恢复条件满足 → 可恢复仓位 |
