### S1-01: Google Search Console 注册（10 分钟）

#### 第 1 步：访问 GSC
打开浏览器访问：
```
https://search.google.com/search-console
```

#### 第 2 步：登录账号
- 使用 Google 账号：`s791400458@gmail.com`
- 点击"立即开始"

#### 第 3 步：添加网站
- 选择"URL 前缀"（不是域名）
- 输入：`https://singclaw.xyz`
- ⚠️ 必须带 `https://` 前缀
- 点击"继续"

#### 第 4 步：验证所有权
- 选择"HTML 标记"
- 复制 `<meta>` 标签，格式如下：
```html

```
- **立即把这串代码发给柯维**，柯维会 5 秒注入到首页

#### 第 5 步：提交 Sitemap
- 验证成功后，左侧菜单点击"站点地图"
- 输入：`sitemap.xml`
- 点击"提交"

- Sitemap 状态显示"成功"

### S1-02: Umami 统计注册（5 分钟）

#### 第 1 步：访问 Umami
打开浏览器访问：
```
https://cloud.umami.is
```

#### 第 2 步：注册/登录
- 点击"Get Started"或"Sign Up"
- 推荐使用 GitHub 账号登录（最快）
- 或用邮箱注册

#### 第 3 步：创建网站
- 登录后点击"+ Add Website"
- 填写信息：
  - **Name**: SingClaw
  - **Domain**: singclaw.xyz
- 点击"Create"

#### 第 4 步：获取 Website ID
- 创建成功后，点击网站名称进入详情
- 找到"Data Website ID"或"Tracking Code"
- 复制那串 UUID，格式如下：
```
xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```
- **立即把这串 ID 发给柯维**，柯维会 5 秒注入统计代码

#### ✅ 完成标志
- Umami 后台显示 singclaw.xyz
- 能看到 Tracking Code
- 柯维确认代码已注入

## 合约部署 - Base 测试网

### 第 1 步：领取测试 ETH（5 分钟）

#### 访问水龙头
打开浏览器访问：
```
https://www.alchemy.com/faucets/base-sepolia
```

#### 连接钱包
1. 点击右上角"Connect Wallet"
2. 选择你的钱包（MetaMask / WalletConnect / Coinbase Wallet）
3. **切换到 Base Sepolia 网络**（重要！）

#### 添加 Base Sepolia 网络（如没有）
- 打开 MetaMask
- 点击右上角网络下拉框
- 选择"Add Network"
- 手动添加：
```
Network Name: Base Sepolia
RPC URL: https://sepolia.base.org
Chain ID: 84532
Symbol: ETH
Block Explorer: https://sepolia.basescan.org
```

#### 领取 ETH
1. 在输入框粘贴钱包地址：
```
0xeF1698B5A6E2AA760db7FE1C34220d127442daE5
```
2. 点击"Send Me ETH"
3. 等待 1-2 分钟到账

#### 确认到账
访问区块浏览器：
```
https://sepolia.basescan.org/address/0xeF1698B5A6E2AA760db7FE1C34220d127442daE5
```
看到 Balance > 0 即成功。
