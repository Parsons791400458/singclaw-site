Layer 1 · Gate Check

避雷筛查 · 12项速查

任何币在深度分析前必须先过这一关。3个以上红旗 = 直接 PASS，不浪费时间。

1流通率 > 60%
低流通 = 未来大量解锁砸盘。< 20% 直接 PASS。
2单次解锁 < 流通量 2%
大额解锁 = 定时炸弹。> 5% = 高危。HEMI 每月解锁 12.3%，8次全跌。
3OI/MC < 30%
OI/市值 = 杠杆程度。> 50% = 赌场结构，> 100% = 极端危险。HEMI 达到 125%。
4FDV/MC < 3倍
完全稀释估值 vs 当前市值。> 5倍 = 未来严重稀释。
5Top 10 持仓 < 40%
头部集中度过高 = 控盘币。> 80% = 庄家说了算。分仓特征（差异 < 1%）= 跑。
6TVL 稳定或增长
锁仓量 = 真实资金沉淀。从高点跌 > 80% = 空心壳。HEMI TVL 从 $4.4亿暴跌 97%。
7DEX 流动性 > $500万
流动性太低 = 买得进卖不出。< $100万 = 不参与。
8有通缩/Buyback
burn + buyback = 供应持续减少。VVV 模式：收入回购销毁 → 价格飞轮。
9有真实产品和用户
能在 DefiLlama/DappRadar 查到真实使用数据。纯 PPT 项目 = 高危。
10上线 > 6个月
新币波动极大。< 3个月 + ICO后一路跌 = 不碰。
117天涨幅 < 100%
暴涨后追高 = 接盘。> 100% 且无基本面催化剂 = 追高陷阱。
12社交情绪 > -0.3
社区已放弃的项目很难起死回生。< -0.5 = 市场已放弃。
🟢 0 红旗 = 安全 🟡 1-2 红旗 = 谨慎 🔴 3-4 红旗 = 高危 💀 5+ 红旗 = 别碰

Layer 4 · Entry Decision

开仓决策 · 杠杆 · 仓位

全条件满足才开仓。杠杆大小由标的特征决定，不是由贪心决定。

✅ 做多信号（至少 4/6）

  • OI 稳步增加 + 价格涨
  • 费率中性或微负（未过热）
  • 大户多空比 > 1.5
  • 空头爆仓增加（逼空中）
  • 交易所余额减少（提币潮）
  • FNG < 30（恐惧中买入）

🔴 跑路信号（至少 4/6）

  • OI 暴增但价格不涨
  • 费率 > 0.1%（多头过热）
  • 散户极度看多（反向指标）
  • 多头爆仓激增
  • 交易所余额暴增（充币卖）
  • FNG > 80（贪婪中卖出）
