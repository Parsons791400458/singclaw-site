# 🎨 三国虾 AI 绘图提示词（Midjourney/Stable Diffusion）

**使用方法**: 复制每个 prompt 到 Midjourney 或 SD，生成后保存为 `shrimp-[角色].png`

---

## 通用参数（Midjourney）
```
--ar 1:1 --v 6 --style raw --q 2
```

## 通用参数（Stable Diffusion）
```
Steps: 30, CFG: 7, Sampler: DPM++ 2M Karras
Positive: masterpiece, best quality, flat illustration
Negative: 3d, realistic, photo, ugly, deformed
```
